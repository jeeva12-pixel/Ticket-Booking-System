# Bus Ticket Booking System

This is a simple **Bus Ticket Booking System** built using Python.  
It allows users to **enter ticket price, number of tickets, and calculate the total amount**.  
The program runs in the console.

## Features
- Enter ticket price  
- Enter number of tickets  
- Calculate total price  
- Option to book multiple tickets  

## Technologies Used
- Python

1. Clone or download the repository.
2. Open `bus_ticket_booking.py` in a Python IDE or VS Code.

4. 4. Follow the on-screen instructions to book tickets.
